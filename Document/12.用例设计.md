# 用例设计

该用例为用户提供本地的Mesh文档，然后用户需要：申请读取文件-> 提供文件名 -> 选择数据结构 -> 选择文件管理器开始导入文件 -> 要求打印 -> 获取Mesh结构的内容 -> 初始化Element的值 -> 输入计算公式 -> 启动保存服务 -> 启动运行 -> 关闭保存服务 -> 退出。
